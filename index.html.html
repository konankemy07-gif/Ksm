📋 INSTALLATION DE L'EA SUR MT5 PC
1
Ouvre MT5 → Fichier → Ouvrir le dossier des données
2
Copie TradingJournalExport.mq5 dans le dossier MQL5/Experts/
3
Dans MT5, appuie sur F5 pour recompiler, puis glisse l'EA sur n'importe quel graphique
4
L'EA exporte trades_export.csv dans MQL5/Files/ toutes les 10 secondes
5
Transfère le CSV sur ton téléphone (WhatsApp, email, Google Drive) et importe-le ici
⚠ NOTE IMPORTANTE
L'émotion ne peut pas être détectée automatiquement par MT5. Les trades importés auront "Neutre" par défaut. Tu peux les modifier manuellement dans Mes Trades.
